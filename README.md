## Application Overview

Pacifist serves <a href="#">Mac administrators</a>, <a href="#">developers</a>, and power users who need package-level access that standard macOS tools don't provide. CharlesSoft's focused utility addresses specific needs that become essential once you encounter them.

<a href="#">System file recovery</a> extracts files from <a href="#">macOS installer</a> packages when system files become corrupted or accidentally deleted. Recover specific system components without complete macOS reinstallation by extracting needed files from installer packages.

<a href="#">Kernel extension access</a> retrieves <a href="#">kext files</a> and other system extensions from installer packages for examination or manual installation. The access serves troubleshooting scenarios where automated installation fails or manual placement is required.

<a href="#">Receipt inspection</a> examines installation receipts showing what previous installations placed on your system. Understand what software modified and where files were installed through receipt analysis capabilities.

<a href="#">Verification features</a> compare installed files against package contents to identify modifications or corruption. Verify installation integrity and detect unauthorized changes through comparison functionality.

<a href="#">Command-line interface</a> enables scripted extraction for automation scenarios. Integrate Pacifist into administrative workflows through command-line access that automation requires.

## Core Capabilities

- <a href="#">PKG file extraction</a> with complete content browsing and selective file retrieval
- <a href="#">DMG and disk image</a> access without mounting through direct content browsing
- <a href="#">Archive format support</a> including ZIP, TAR, and compressed variants
- <a href="#">macOS installer package</a> inspection for system file recovery
- <a href="#">Installation verification</a> comparing installed files against package contents
- <a href="#">Hierarchical browsing</a> displaying package structure clearly
- <a href="#">Batch extraction</a> of multiple files or complete directories
- <a href="#">Receipt analysis</a> showing previous installation details
- <a href="#">Command-line tools</a> for scripted and automated extraction
